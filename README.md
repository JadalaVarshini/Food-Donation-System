# Food-Donation-System

## Overview
The Food Donation System is a web-based application developed to reduce food wastage by connecting food donors with people in need. This platform helps donors donate excess food easily and enables receivers or NGOs to access available food resources.

## Features

- User Registration and Login
- Food Donation Management
- Donor Dashboard
- Receiver Dashboard
- Admin Panel
- Food Availability Tracking
- Responsive User Interface
- Easy Navigation

## Technologies Used

### Frontend
- HTML
- CSS
- JavaScript

### Backend
- Python
- Flask

### Database
- SQLite / MySQL

## Project Objective

The main objective of this project is:

- To reduce food wastage
- To help needy people access food
- To provide a simple online food donation platform
- To improve food distribution efficiency

## Installation Steps

### Step 1: Clone Repository

```bash
git clone https://github.com/JadalaVarshini/Food-Donation-System.git
```

### Step 2: Open Project Folder

```bash
cd Food-Donation-System
```

### Step 3: Install Dependencies

```bash
pip install -r requirements.txt
```

### Step 4: Run Application

```bash
python app.py
```
